Installing Intel Network Drivers for Linux*

Intel provides the following packages for managing Intel® PRO/100 and PRO/1000 Families of Adapters in Linux* operating systems.

Base Drivers
For base driver features,  installation, configuration, and command line parameter information, refer to:

For PRO/100 adapters:  e100.htm
For PRO/1000 adapters:  e1000.htm

Advanced Network Services (ANS)
If you are setting up teams or VLANs, refer to the instructions for Intel Advanced Network Services (iANS) in ans.htm.

Configuration Utility
Intel base drivers and iANS for Linux can be configured with the Intel® Linux LAN Adapters PROCfg Application. This utility only works on Intel 32-bit architectures running Linux. For more information, refer to procfg.htm.

Simple Network Management Protocol (SNMP)
Intel provides a Simple Network Management Protocol (SNMP) agent extension for Linux systems. This agent provides both general and advanced information on the Intel adapters in the system.

Intel Support for Linux

For general information and support, see the Support page.

If an issue is identified with the released source code on the supported kernel with a supported adapter, email the specific information related to the issue to linux.nics@intel.com.
