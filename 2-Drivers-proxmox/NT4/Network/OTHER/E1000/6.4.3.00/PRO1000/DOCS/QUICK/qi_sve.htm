Intel® PRO Network Adapter installation

Revision 04/2004

Installera Intel PCI-, PCI-X- eller PCI-Express*-kortet

  1. Stäng av datorn och koppla ifrån nätkabeln.

  2. Ta bort datorns hölje och kortplatsluckan från en kortplats som kan användas för ditt kort.

  3. För PCI Express-kort: sätt in kortets kantkontakt i PCI Express-platsen och fäst kortklammern vid höljet.
    För PCI- och PCI-X-kort: för bestämt in kortets kantkontakt i PCI-platsen tills det är helt intryckt. Fäst kortklammern vid höljet.

  4. Sätt tillbaka datorns hölje. Anslut strömkabeln.

Ansluta nätverkskabeln

Intel® PRO/100 och PRO/1000 kopparkort: Anslut kabeln eller kablarna till kortet med en TPE-nätverkskabel (Twisted Pair Ethernet) av kategori 3, 4 eller 5 för 10 Mbps-nätverk. Använd kabel av kategori 5 (4 par) för 100 eller 1000 Mbps-nätverk och för alla nätverk i boendemiljö.

Intel® PRO/1000 fiberkort med SC-kontakt: Ta bort och spara luckan till fiberoptikanslutningen. För in en fiberoptikanslutning av 1000Base-SX duplextyp SC i TX/RX-portarna på kortklammern.

Intel® PRO/1000 fiberkort med LC-kontakt: Ta bort och spara luckan till fiberoptikanslutningen. För in en LC-kontakt i kortet, med korrekt kablage för din korttyp (1000BASE-SX eller 1000BASE-LX). Omvandlingskablar till andra anslutningstyper (t.ex. SC) kan användas om kablarna överensstämmer med kortets optiska specifikationer, inklusive längdbegränsningar.

 Starta datorn och följ instruktionerna nedan för drivrutinsinstallation.

Installera drivrutiner och tillägg

Obs! Intel PRO/1000-serverkort fungerar inte med operativsystemen Windows* 95, Windows 98* och Windows Me*.

Windows-operativsystem (utom Windows NT* 4.0)

Observera att du måste ha administrativa rättigheter för att kunna installera drivrutinerna.

  1. Kortet identifieras och Windows installerar antingen en drivrutin som redan finns i Windows eller startar Guiden ny maskinvara.

  2. Om guiden startar följer du instruktionerna för hur du installerar drivrutinen från den medföljande cd:n.

  3. Om en drivrutin som redan finns i Windows installeras (guiden visas inte) uppdaterar du drivrutinen genom att sätta in cd-skivan från Intel och klicka på knappen Installera drivrutiner när fönstret Intel PRO-nätverksanslutningar visas.

  4. I dialogrutan Installationsalternativ väljer du Advanced Network Services om du behöver konfigurera team eller VLAN.

Windows NT 4.0

Obs! Programvaran Intel PROSet installeras automatiskt vid installationen av kortets drivrutiner.

  1. Starta Windows NT 4.0 och dubbelklicka på nätverksikonen i Kontrollpanelen. Klicka på fliken Kort och klicka sedan på Lägg till.

  2. När en lista över kort visas matar du in cd-skivan som medföljde ditt Intel-kort eller väljer andra media i stället för att välja ett kort från listan. Klicka sedan på Diskett finns.

  3. Om autorun-dialogrutan visas stänger du den och fortsätter.

  4. Ange sökvägen, klicka på OK och fortsätt sedan installationen. När installationen är färdig visas det nya Intel-kortet i listan över Nätverkskort.

  5. Klicka på Stäng för att avsluta.

  6. Starta om datorn.

Linux*

Du kan bygga ett binärt RPM*-paket för denna drivrutin genom att köra ”rpmbuild -tb <filnamn.tar.gz>”. Ersätt <filnamn.tar.gz> med det specifika namnet för drivrutinen.

  ANMÄRKNINGAR: För att bygget ska fungera ordentligt måste den kernel som för närvarande körs överensstämma med versionen och konfigurationen för de installerade kernel-källorna. Om du just har kompilerat om din kernel ska du starta om datorn nu.

RPM-funktionen har endast testats med Red Hat-versioner.

För Intel PRO/100-kort ska du använda e100 som drivrutinsnamn.

  1. Flytta basdrivrutinens tar-fil till den katalog du väljer. Använd till exempel ”/hem/användarnamn/e1000” eller ”/usr/local/src/e1000”.

  2. Packa upp filen (<x.x.x> anger versionsnumret för drivrutinsfilen):

    tar zxf e1000-<x.x.x>.tar.gz

  3. Ändra till drivrutinens källkatalog, där <x.x.x> anger versionsnumret för drivrutinsfilen:

    cd e1000-<x.x.x>/src/

  4. Kompilera drivrutinsmodulen:

    skapa installerare

    Binärfilen installeras som:

    /lib/modules/<kernel_version>/kernel/drivers/net/e1000.o

    De installationsplatser som anges ovan är standardplatserna. De kanske inte stämmer för vissa versioner av Linux. Mer information finns på sidan ldistrib.htm.

  5. Installera modulen:

    insmod e1000 <parameter>=<värde>

  6. Tilldela en IP-adress till gränssnittet genom att ange följande, där <x> är gränssnittsnumret:

    ifconfig eth<x> <IP_address>

  7. Verifiera att gränssnittet fungerar. Skriv in följande, där <IP_address> är en IP-adress för en annan maskin i samma delnät som det gränssnitt som testas:

    ping <IP_address>
