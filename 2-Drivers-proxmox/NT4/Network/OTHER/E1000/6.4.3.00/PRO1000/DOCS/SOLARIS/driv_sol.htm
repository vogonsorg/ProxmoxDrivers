Overview

Versions Supported: Solaris* 7, or Solaris 8 Update 2 operating system for x86 architecture, Solaris 9 for x86 architecture, with latest patch clusters/patches for respective operating system versions for x86.

NOTE: If the existing driver version is 2.000.0 or lower, completely uninstall the drivers and then install the newer version. Do not upgrade the 1.xxx.x version with the 2.xxx.x version as both package formats are slightly different. The new format does not configure adapters and uses different scripts than the previous package. New device IDs are also added in the new package.

Read the instructions to install the Solaris 7 or Solaris 8 driver package for the Intel adapter. You must have administrative rights to successfully install the driver.

The INTCGigaE package is available in two formats:

  • Datastream format (INTCGigaE.pkg).

  • Compressed and tar File system format (INTCGigaE.tar.Z).

If you already have a previous version of this package, remove it first before installing this package. Use 'pkgrm' to remove a package and 'pkgadd' to add a package.
