Installing the Driver

Before installing the drivers, insert the adapter in the computer and plug in the network cable. When Windows 2000 discovers the new adapter, the Found New Hardware Wizard automatically starts. If you are installing just one adapter and do not intend on installing more adapters, follow the instructions below. Otherwise, click here.

  1. If you are installing drivers from the Product CD, insert the CD in the CD-ROM drive. If the autorun program on the CD displays, ignore it.

  2. In the Wizard Welcome screen, click Next.

  3. Choose the Search option and click Next.

  4. If you are installing drivers from the Product CD, click the CD-ROM checkbox on the Locate Driver screen. Make sure that it is the only checkbox selected.

    If you downloaded drivers from the support website, click the Specify Location checkbox instead and browse to the directory containing the drivers.

  5. Click Next.

  6. Windows 2000 searches for a driver. When the search is complete, a message indicates a driver was found on the CD. Click Next.

  7. The necessary files are copied to your computer. The wizard displays a Completed message. 

  8. Click Finish.

NOTE: You can now install Intel PROSet to configure advanced settings and set up teams, VLANs, and other features.

Installing Additional Adapters or more than One Adapter

To install additional adapters or more than one adapter, follow the instructions below.

  1. Insert the Product CD in the CD-ROM drive. The CD autorun program starts. 

    If you downloaded drivers from the support website, browse to the directory containing the downloaded files and extract the files. The  autorun automatically starts.

  2. Click Install Base Driver. A message telling you that a reboot is required on Windows 98 and Windows NT 4.0 systems. 

  3. Click OK to install all base drivers.

There are no special instructions for installing drivers of non-Intel adapters, for example, for multi-vendor fault tolerance. Follow the instructions that are shipped with that adapter. 

Advanced Settings for Windows

To view or change advanced settings for your driver/adapter, use Intel PROSet. Advanced settings are described in the Intel PROSet Help. For instructions on installing Intel PROSet, click here.

CAUTION: Only expert users or network administrators should change advanced settings. You may severely affect your adapter's operation by changing a parameter to a value that is not compatible with your network.

Updating the Driver

You can update drivers using either the Product CD or the Update Device Driver Wizard.

Updating with the Product CD (Recommended)

You can update drivers using the autorun program on the Product CD. Inr3l recommends you follow this process.

  1. Insert the Product CD in the CD-ROM drive. The CD autorun program starts.

    If you downloaded drivers from the support website, browse to the directory containing the downloaded files and extract the files. The  autorun automatically starts.

  2. Click Install Base Driver. A message telling you that a reboot is required on Windows 98 and Windows NT 4.0 systems. 

  3. Click OK to update the drivers.

Updating with the Update Device Driver Wizard

  1. From the Control Panel, double-click the System icon, select the Hardware tab, and click the Device Manager button.


  2. Double-click Network Adapters and right-click the Intel adapter listing to display its menu. 

  3. Click the Properties menu option.

  4. In the Properties dialog box, click the Driver tab and click Update Driver. The Update Device Driver Wizard displays. Click Next.

  5. At the prompt "What do you want the wizard to do?", select the Search Option and click Next.

  6. Select the adapter type. Click Next.

  7. Click Finish.

If you are installing a driver in a computer with existing Intel adapters, be sure to update all the adapters and ports with the same new software by following the steps above for each adapter. This procedure ensures that all adapters function correctly.

For troubleshooting information, click Troubleshooting in the Table of Contents.

Installing Intel PROSet

  1. Insert the Product CD. The Autorun program starts. 

    If you downloaded drivers from the support website, extraction of the files automatically starts the autorun.

  2. Click Install Software. A message box appears asking if you want to continue. You can use Intel PROSet software to configure advanced settings, set up teams and VLANs, and use other features.

Removing Adapters and Software

This section describes the following scenarios for removing an adapter:

  • Uninstall Intel PROSet

  • Temporarily disable an adapter

  • Replace adapter

  • Remove an adapter in a multi-adapter environment

  • Remove the sole adapter (complete removal of adapter and software)

NOTE: If your computer is a hot plug-capable system, see your computer documentation for special removal instructions.
CAUTION: Removing an adapter results in a disruption of all network traffic through that adapter.

Uninstall Intel PROSet

You do not need to uninstall Intel PROSet unless you are permanently removing all Intel adapters. You uninstall Intel PROSet using either Add-Remove Programs or the Product CD.

NOTE:  Before uninstalling Intel PROSet, you must remove teams and VLANs configured in Intel PROSet.  

Uninstalling Intel PROSet with the Product CD

  1. Insert the Product CD in the CD-ROM drive. The CD autorun program starts. 

  2. Click Install Software. The InstallShield Wizard screen displays.

  3. Click Next. The Program Maintenance dialog displays.

  4. Click Modify. The Custom Setup dialog displays.

  5. Click the icon next to the feature you want to remove. From the pop-up dialog, click Do not install this feature. Click Next.

  6. Click Install.

  7. Click Finish.

The installer removes Intel PROSet from the computer. 

Uninstalling with Add-Remove Programs 

  1. From the Control Panel, double-click Add-Remove Programs. The Add-Remove Programs window displays.

  2. Select Intel(R) PROSet. Click Change/Remove. 

  3. When the confirmation message displays, click Yes. 

  4. Allow the uninstall process to complete. Click Finish.

Temporarily Disable an Adapter

If you are testing your system to determine network faults, particularly in a multi-adapter environment, it is recommended that you temporarily disable an adapter. 

  1. Open Intel PROSet.

  2. Right-click the adapter name and select Remove Adapter from the drop-down list. The driver for this adapter is immediately uninstalled, and no network traffic passes through it. 

When the computer is restarted, the drivers automatically reload and network communications resume. 

You can also re-enable the adapter without restarting. Follow the steps for installing the drivers, but use the Repair option if you reinstall drivers with the Product CD. 

Replace Adapter

After installing an adapter in a specific slot, Windows treats any other adapter of the same type as a new adapter. If you remove the installed adapter and insert it into a different slot, Windows recognizes it as a new adapter. Make sure that you follow the instructions below carefully.

  1. Open Intel PROSet.

  2. If the adapter is part of a team, right-click the adapter name and select Remove from Team. 

  3. Right-click the adapter you want to remove, and select Remove Adapter from the drop-down menu. 
    NOTE: If you do not perform this instruction, you see a "phantom adapter" when you restart. The driver and registry association exist, but there is no physical adapter. 

  4. Shut-down the computer and unplug the power cable.

  5. Disconnect the network cable from the adapter.

  6. Open the case and remove the adapter. 

  7. Insert the replacement adapter, and secure it with a screw if necessary. Use the same slot, otherwise Windows 2000 assumes that there is an adapter.

  8. Reconnect the network cable. 

  9. Close the case, reattach the power cable, and power-up the computer. 

  10. Open Intel PROSet and check to see that the adapter is available.

  11. If the former adapter was part of a team, right-click the adapter name, and select Add to Team from the drop-down menu, then specify the team from the secondary drop-down menu. 

  12. If the former adapter was tagged with a VLAN, you must tag the new adapter in a similar fashion.

Uninstall the Driver

You should uninstall the driver if you are permanently removing adapters, or if you need to perform a clean installation of new drivers.

Before you uninstalling the driver, 

  • If the adapter is a member of a team or VLAN, remove the adapter from the team or VLAN in Intel PROSet. 
  • Uninstall Intel PROSet.

To uninstall the driver (recommended):

  1. From the Control Panel, double-click Add/Remove Programs. 

  2. Select Intel PRO Ethernet Adapter and Software. 

  3. Click Change/Remove. 

  4. When the confirmation dialog displays, click Yes. 

  5. Allow the uninstall process to complete. Click Finish.

NOTE: Creating an Express Team automatically installs any adapters that were uninstalled but still physically present in a system.
NOTE: If you uninstall a port of an adapter that is associated with an Express Team, the Express Team will be removed.
