Network Adapters with Fiber SC Connections

CAUTION: The fiber optic ports contain a Class 1 laser device. When the ports are disconnected, always cover them with the provided plug. If an abnormal fault occurs, skin or eye damage may result if in close proximity to the exposed ports.

Remove and save the fiber optic connector cover. Insert a fiber optic cable into the ports on the network adapter bracket as shown. 

Most connectors and ports are keyed for proper orientation. If the cable you are using is not keyed, check to be sure the connector is oriented properly (transmit port connected to receive port on the link partner, and vice versa).

All Intel fiber-based adapters with SC connectors support the 1000Base-SX connectivity standard. The connection requirements are as follows:

  • Laser wavelength: 850 nanometer (not visible). 

  • SX Cable type:

    • Multi-mode fiber with 50 micron core diameter; maximum length is 550 meters.

    • Multi-mode fiber with 62.5 micron core diameter; maximum length is 275 meters.

  • Connector type: SC.

  • Maximum cable length: 300 meters. 

The adapter must be connected to a compatible link partner, such as an IEEE 802.3z-compliant gigabit switch, which is operating at the same laser wavelength as the adapter.

Network Adapters with Fiber LC Connections

CAUTION: The fiber optic ports contain a Class 1 laser device. When the ports are disconnected, always cover them with the provided plug. If an abnormal fault occurs, skin or eye damage may result if in close proximity to the exposed ports.

Remove and save the fiber optic connector cover. Insert a fiber optic cable into the ports on the network adapter bracket as shown. 

Most connectors and ports are keyed for proper orientation. If the cable you are using is not keyed, check to be sure the connector is oriented properly (transmit port connected to receive port on the link partner, and vice versa).

Most Intel fiber-based adapters support the 1000Base-SX connectivity standard. The connection requirements are as follows:

  • Laser type: 850 nanometer wavelength (not visible).

  • SX Cabling: 

    • Multi-mode fiber with 50 micron core diameter. Maximum length is 550 meters.

    • Multi-mode fiber with 62.5 micron core diameter. 
      Maximum length is 275 meters

  • Connector type: LC. 

The adapter must be connected to a compatible link partner, such as an IEEE 802.3z-compliant gigabit switch, which is operating at the same laser wavelength as the adapter. 

"LX" Adapters

Adapters with an "LX" marking on the bracket support the 1000Base-LX connectivity standard. They are designed for long-haul operation, and have different connectivity requirements:

  • Laser type: 1310 nanometer wavelength (not visible).

  • LX Cabling: single-mode fiber with 9.0 micron core diameter. 

  • Maximum length: 10 kilometers. 

  • Connector type: LC. 

(NOTE: Although LX adapters are designed for use with long-haul single-mode fiber, they can also be used with short-haul multi-mode fiber. Maximum cable length is 550 meters. Must be connected to compatible LX ports.)
