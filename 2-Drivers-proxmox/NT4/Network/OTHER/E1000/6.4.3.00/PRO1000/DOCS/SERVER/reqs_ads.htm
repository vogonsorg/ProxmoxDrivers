Compatibility Notes

Intel's first generation PRO/1000 Gigabit Server Adapter has the following limitations:

  • VLANs on this adapter are supported only in Microsoft Windows (NT 4.0, 2000, XP and Windows Server 2003).

  • Jumbo Frames on this adapter are not supported under Linux or Solaris. 

  • This adapter operates correctly in 32- and 64-bit slots. However, when inserted in a 64-bit PCI slot, it will incorrectly indicate that it is in a 32-bit slot.

  • This adapter does not support packet tagging for QoS or VLANs.
