Push Install Instructions for Microsoft* Windows* XP

The following document provides instructions for a basic unattended installation of Microsoft* Windows* XP that includes the installation of drivers for the Intel® PRO/100 and Intel® PRO/1000 adapters. A "Push", or unattended installation of Windows XP provides a means for network administrators to easily install Windows XP on systems that have similar equipment.

For more information and instructions, see the pushxp.txt file in the APPS\SETUP\PUSH\WS03XP32 directory on the Product CD.
