Instalação do adaptador de rede Intel® PRO

Revisão 04/2004

Instalar o adaptador Intel PCI, PCI-X ou PCI-Express*

  1. Desligue o computador e desconecte o fio de eletricidade.

  2. Remova a tampa do computador e a tampa do slot correspondente ao seu adaptador.

  3. Para adaptadores PCI Express: insira a borda do conector no slot PCI Express e prenda a braçadeira ao chassi.
    Para adaptadores PCI e PCI-X: empurre firmemente a borda do adaptador no slot PCI até que ele esteja completamente inserido. Prenda a braçadeira ao chassi.

  4. Recoloque a tampa do computador. Ligue o fio de eletricidade.

Conecte o cabo de rede

Adaptadores Intel® PRO/100 e PRO/1000 de cobre: Conecte o(s) cabo(s) de rede ao adaptador usando cabos de par trançado Ethernet (TPE) Categoria 3, 4 ou 5 para redes de 10 Mbps. Use cabos categoria 5 (4 pares) para as redes de 100 ou 1000 Mbps e para as redes residenciais.

Adaptadores de fibra Intel® PRO/1000 com conector SC: Remova e guarde a tampa do conector de fibra ótica. Insira um conector de fibra ótica SC do tipo 1000Base-SX duplex nas portas TX/RX da braçadeira do adaptador.

Adaptadores de fibra Intel® PRO/1000 com conector LC: Remova e guarde a tampa do conector de fibra ótica. Insira um conector LC no adaptador com o cabeamento correto para o seu tipo de adaptador (1000BASE-SX ou 1000BASE-LX). Podem ser usados cabos de conversão para outros tipos de conectores (como, por exemplo, SC) se o cabeamento corresponder às especificações óticas do adaptador, inclusive as limitações de comprimento.

 Ligue o computador e siga as instruções de instalação do driver.

Instalar os drivers e extensões

Nota: Os adaptadores de servidor Intel PRO/1000 não funcionam com os sistemas operacionais Windows* 95, Windows 98* e Windows Me*.

Sistemas operacionais Windows (exceto Windows NT* 4.0)

Note que você deve ter direitos de administrador para o sistema operacional instalar os drivers.

  1. O adaptador é detectado e o Windows instala um driver residente do Windows ou inicia o assistente de Novo hardware encontrado.

  2. Se esse assistente iniciar, siga as orientações para instalar o driver a partir do CD nesse pacote.

  3. Se for instalado um driver residente do Windows (o assistente não aparece), atualize o driver inserindo o CD da Intel e clicando no botão Instalar os drivers quando aparecer a janela de execução automática Conexões de rede Intel PRO.

  4. Quando o diálogo Instalar opções aparecer, escolha Serviços avançados de rede se necessitar configurar grupos ou VLANs.

Windows NT 4.0

Nota: o software do utilitário Intel PROSet é instalado automaticamente durante a instalação do driver do adaptador.

  1. Inicie o Windows NT 4.0, clique duas vezes no ícone Rede no Painel de controle, clique na guia Adaptadores e clique em Adicionar.

  2. Quando a lista de adaptadores aparecer, em vez de escolher um adaptador da lista, insira o CD fornecido com o seu adaptador ou aponte para a outra mídia e clique em “Com disco”.

  3. Se aparecer a tela de Execução automática do CD, feche-a e continue.

  4. Especifique o caminho, clique em OK e continue a instalação. Ao completar a instalação, aparecerá o novo adaptador Intel na lista de Adaptadores de Rede.

  5. Clique em Fechar para terminar.

  6. Reinicialize o computador.

Linux*

Para compilar um pacote RPM* binário deste driver execute 'rpmbuild -tb <filename.tar.gz>'. Substitua <filename.tar.gz> com o nome de arquivo específico do driver.

  NOTAS: Para que a compilação funcione adequadamente, o kernel sendo executado no momento DEVE coincidir com a versão e configuração das fontes do kernel instaladas. Se já tiver recompilado o kernel, reinicialize agora o sistema.

A funcionalidade do RPM foi testada somente nas distribuições do Red Hat.

Para os adaptadores Intel PRO/100 use e100 como o nome do driver.

  1. Mova o arquivo tar do driver básico para o diretório de sua escolha. Por exemplo, use '/home/username/e1000' ou '/usr/local/src/e1000'.

  2. Para efetuar o untar/unzip do arquivo, onde <x.x.x> é o número da versão do arquivo tar do driver:

    tar zxf e1000-<x.x.x>.tar.gz

  3. Mude para o diretório src do driver, onde <x.x.x> é o número da versão do tar do driver:

    cd e1000-<x.x.x>/src/

  4. Compile o módulo do driver:

    make install

    O binário será instalado como:

    /lib/modules/<kernel_version>/kernel/drivers/net/e1000.o

    Os locais de instalação listados acima são os locais padrão. Eles podem não estar corretos para certas distribuições do Linux. Para mais informações vá para ldistrib.htm.

  5. Instalar o módulo:

    insmod e1000 <parâmetro>=<valor>

  6. Atribuir um endereço IP para a interface digitando o seguinte, onde <x> é o número da interface:

    ifconfig eth<x> <endereço_IP>

  7. Verifique se a interface funciona. Digite o seguinte, onde <endereço_IP> é o endereço IP para uma outra máquina na mesma subrede que a interface que está sendo testada:

    ping <endereço_IP>

Outros sistemas operacionais

Para instalar outros drivers, consulte o Guia do usuário no CD do adaptador ou visite o site de suporte: http://support.intel.com
