Installing SNMP on Microsoft* Windows* 2000, XP and Windows Server 2003

To install SNMP, start the autorun menu from the Product CD. Select the type of adapter and then click Install SNMP. Follow the instructions on the screen.

The Simple Network Management Protocol (SNMP) is a protocol for managing networks via messages (called "Traps") sent from SNMP-compliant network components and collected by one or more servers. This utility should only be employed by experienced network administrators. Additional software/services must be installed on your network prior to installing the Intel® SNMP agent.
