PCI Hot Plug Support for UnixWare

Intel network adapters are enabled for use in selected servers equipped with PCI Hot Plug support and running UnixWare.

NOTE: If you replace an adapter in a Hot Plug slot, do not place the removed adapter back into the same network until the server has rebooted. The two adapters may have the same virtual MAC address and create a conflict.
NOTE: PCI Hot Plug only works between identical Intel network adapters.

For driver installation and configuration see the UnixWare driver instructions.

Software Requirements:

  • UnixWare operating system with respective Hot Plug patch

  • Compatible Hot Plug driver

  • DDI8 driver

Hardware Requirements:

  • Hot Plug-capable server with latest BIOS installed to support Hot Plug.
