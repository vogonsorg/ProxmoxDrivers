Installation des cartes réseau Intel® PRO

Révision 04/2004

Installer une carte Intel PCI, PCI-X ou PCI-Express*

  1. Mettez l'ordinateur hors tension et débranchez le cordon d'alimentation.

  2. Retirez le panneau de l'ordinateur et le panneau du connecteur de carte correspondant à votre carte.

  3. Pour les cartes PCI Express : insérez le connecteur au bord de la carte dans le connecteur PCI Express, et attachez solidement le support sur le châssis.
    Pour les cartes PCI et PCI-X : insérez fermement le connecteur au bord de la carte dans le connecteur PCI, jusqu'à ce qu'il soit complètement logé. Attachez solidement le support sur le châssis.

  4. Remettez le panneau de l'ordinateur en place. Branchez le cordon d'alimentation.

Brancher le câble réseau

Cartes cuivre Intel® PRO/100 et PRO/1000 : Branchez le(s) câble(s) réseau sur la carte en utilisant une paire torsadée Ethernet de catégorie 3, 4 ou 5 pour les réseaux 10 Mbit/s. Utilisez un câble de catégorie 5 (4 paires) pour les réseaux 100 Mbit/s ou 1000 Mbit/s, ainsi que pour tout réseau résidentiel.

Cartes fibre Intel® PRO/1000 avec connecteur SC : Retirez le panneau du connecteur à fibre optique et mettez-le sur le côté. Insérez un connecteur SC à fibre optique de type 1000Base-SX duplex dans les ports TX/RX du support de la carte.

Cartes fibre Intel® PRO/1000 avec connecteur LC : Retirez le panneau du connecteur à fibre optique et mettez-le sur le côté. Insérez un connecteur LC dans la carte en utilisant un câblage approprié pour le type de carte (1000BASE-SX ou 1000BASE-LX). Les câbles de conversion pour d'autres types de connecteur (p. ex., un connecteur SC) peuvent être utilisés si le câblage est conforme aux spécifications optiques de la carte, y compris les limites de longueur.

 Démarrez votre ordinateur et suivez les instructions d'installation des pilotes ci-dessous.

Installer les pilotes et les extensions

Remarque : les cartes Intel PRO/1000 pour serveur ne fonctionnent pas sur les ordinateurs Windows* 95, Windows 98* et Windows Me*.

Systèmes d'exploitation Windows (sauf Windows NT* 4.0)

Notez que vous devez disposer de droits d'administrateur sur le système d'exploitation pour installer les pilotes.

  1. La carte est détectée et Windows installe un pilote Windows résident ou démarre l'Assistant Matériel détecté.

  2. Si l'assistant démarre, suivez les invites pour installer le pilote depuis le CD fourni avec la carte.

  3. Si un pilote Windows résident est installé (l'assistant ne s'affiche pas), mettez le pilote à jour en insérant le CD Intel et en cliquant sur le bouton Installer les pilotes lorsque la fenêtre d'exécution automatique Connexions réseau Intel PRO s'affiche.

  4. Lorsque la boîte de dialogue Options d'installation s'affiche, sélectionnez Services réseau avancés si vous voulez configurer des associations ou des VLAN.

Windows NT 4.0

Remarque : l'utilitaire Intel PROSet est installé automatiquement au cours de l'installation du pilote de la carte.

  1. Démarrez Windows NT 4.0, double-cliquez sur l'icône Réseau du Panneau de configuration, cliquez sur l'onglet Cartes réseau, puis sur Ajouter.

  2. Lorsque la liste des cartes réseau apparaît, insérez le CD livré avec votre carte ou indiquez un autre support plutôt que de choisir une carte dans la liste, et cliquez ensuite sur Disquette fournie.

  3. Si un écran d'exécution automatique apparaît, fermez-le.

  4. Spécifiez le chemin d'accès, cliquez sur OK et continuez l'installation. Lorsque celle-ci est terminée, la nouvelle carte Intel apparaît dans la liste Cartes réseau.

  5. Cliquez sur Fermer pour terminer.

  6. Redémarrez l'ordinateur.

Linux*

Pour créer un package RPM* binaire de ce pilote, exécutez  'rpmbuild -tb <nom_du_fichier.tar.gz>'. Remplacez <nom_du_fichier.tar.gz> par le nom de fichier spécifique du pilote.

  REMARQUE : pour que le module fonctionne correctement, la version et la fonfiguration du noyau actuellement exécuté DOIVENT correspondre aux source du noyau installé. Si vous venez de recompiler le noyau, redémarrez l'ordinateur maintenant.

La fonctionnalité RPM a été testée uniquement sur les distributions Red Hat.

Pour les cartes Intel PRO/100, utilisez e100 comme nom de pilote.

  1. Déplacez le fichier tar du pilote de base dans un répertoire au choix. Par exemple, utilisez "/home/username/e1000" ou "/usr/local/src/e1000".

  2. Déballez/décompressez l'archive, où <x.x.x> correspond à la version du fichier tar du pilote :

    tar zxf e1000-<x.x.x>.tar.gz

  3. Passez au répertoire src du pilote, où <x.x.x> correspond au numéro de version du fichier tar du pilote :

    cd e1000-<x.x.x>/src/

  4. Compilez le module du pilote :

    make install

    Le fichier binaire sera installé sous :

    /lib/modules/<kernel_version>/kernel/drivers/net/e1000.o

    Les emplacements d'installations ci-dessus sont les emplacements par défaut. Il est possible qu'ils ne soient pas corrects pour certaines distributions Linux. Pour de plus amples informations, visitez ldistrib.htm.

  5. Installez le module :

    insmod e1000 <paramètre>=<valeur>

  6. Affectez une adresse IP à l'interface en entrant ce qui suit, où <x> correspond au numéro de l'interface :

    ifconfig eth<x> <adresse_IP>

  7. Vérifiez que l'interface fonctionne. Entrez ce qui suit, où <adresse_IP> correspond à l'adresse IP d'un autre ordinateur du sous-réseau sur lequel l'interface est testée :

    ping <adress_IP>
