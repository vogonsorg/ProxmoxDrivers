인텔® 네트워크 어댑터 설치

개정일: 2004년 4월

Intel PCI, PCI-X 또는 PCI-Express* 어댑터 설치

  1. 컴퓨터를 끄고 전원 코드를 뽑습니다.

  2. 컴퓨터 덮개를 제거한 다음 해당 어댑터와 일치하는 슬롯에서 어댑터 슬롯 덮개를 제거합니다.

  3. PCI Express 어댑터의 경우: 어댑터 모서리 커넥터를 PCI Express 슬롯에 끼우고 브래킷을 섀시에 고정시킵니다.
    PCI 및 PCI-X 어댑터의 경우: 어댑터 모서리 커넥터를 세게 밀어서 PCI 슬롯에 완전히 끼우고 브래킷을 섀시에 고정시킵니다.

  4. 컴퓨터 덮개를 다시 덮고 전원 코드를 연결합니다.

네트워크 케이블 연결

Intel® PRO/100 및 PRO/1000 구리 어댑터: 10Mbps 네트워크용 카테고리 3, 4 또는 5 트위스트 페어 이더넷(TPE) 네트워크 케이블을 사용하여 네트워크 케이블을 어댑터에 연결하십시오. 100 또는 1000 Mbps 네트워크와 주거 지역 네트워크에는 카테고리 5(4 페어) 케이블을 사용하십시오.

SC 커넥터가 있는 Intel® PRO/1000 구리 어댑터: 광섬유 커넥터 덮개를 분리하여 보관하십시오. 1000Base-SX 듀플렉스 유형 SC 광섬유 커넥터를 어댑터 브래킷의 TX/RX 포트에 끼우십시오.

LC 커넥터가 있는 Intel® PRO/1000 구리 어댑터: 광섬유 커넥터 덮개를 분리하여 보관하십시오. 어댑터 종류에 알맞은 케이블을 사용하여 LC 커넥터를 어댑터에 끼웁니다(1000BASE-SX 또는 1000BASE-LX). 길이 제한을 포함한 어댑터의 광학 사양이 케이블 연결 사양과 맞는 경우에는 SC와 같이 다른 커넥터 종류에 사용하는 전환 케이블을 사용할 수도 있습니다.

 컴퓨터를 시작하고 아래의 드라이버 설치 안내를 따르십시오.

드라이버 및 확장 옵션 설치

참고: Intel PRO/1000 서버 어댑터는 Windows* 95, Windows 98* 및 Windows Me* 컴퓨터에서는 작동하지 않습니다.

Windows 운영 체제(Windows NT* 4.0 제외)

드라이버를 설치하려면 운영 체제에 대한 관리자 권한이 있어야 합니다.

  1. 어댑터가 감지되면 Windows가 Windows 상주 드라이버를 설치하거나 새 하드웨어 발견 마법사를 시작합니다.

  2. 마법사가 시작되면 안내 메시지에 따라 CD로부터 이 패키지에 있는 드라이버를 설치합니다.

  3. Windows 상주 드라이버를 설치하는 경우(마법사는 나타나지 않음) Intel PRO 네트워크 연결 자동 실행 창이 나타나면 Intel CD를 넣고 드라이버 설치를 클릭하여 드라이버를 업데이트합니다.

  4. 설치 옵션 대화 상자가 나타날 때, 팀 또는 VLAN을 설정해야 하면 고급 네트워크 서비스를 선택합니다.

Windows NT 4.0

참고: Intel PROSet 유틸리티 소프트웨어는 어댑터 드라이버를 설치할 때 자동으로 설치됩니다.

  1. Windows NT 4.0을 시작한 다음 제어판의 네트워크 아이콘을 두 번 누른 후 [어댑터] 탭을 누르고 [추가]를 누릅니다.

  2. 어댑터 목록이 표시되면 목록에서 어댑터를 선택하는 대신에 어댑터와 함께 제공된 CD를 넣거나 다른 미디어를 가리킨 다음 [디스크 있음]을 누릅니다.

  3. CD 자동 실행 화면이 나타나면 닫고 계속합니다.

  4. 경로를 지정하고 [확인]을 누른 후 설치를 계속합니다. 설치가 끝나면 네트워크 어댑터 목록에 새 Intel 어댑터가 표시됩니다.

  5. [닫기]를 눌러 종료합니다.

  6. 컴퓨터를 다시 시작합니다.

Linux*

이 드라이버의 이진 RPM* 패키지를 빌드하려면 'rpmbuild -tb <filename.tar.gz>'를 실행합니다. <filename.tar.gz>는 해당 드라이버의 고유 파일 이름으로 대체하십시오.

? 주: 빌드가 올바로 작동하려면 이미 설치된 커널 소스의 버전 및 구성이 현재 작동 중인 커널에 맞아야 합니다. 커널을 다시 컴파일했으면 시스템을 다시 부팅하십시오.

RPM 기능은 Red Hat 버전에서만 테스트되었습니다.

Intel PRO/100 어댑터의 경우 드라이버 이름으로 e100을 사용하십시오.

  1. 선택한 디렉토리로 기준 드라이버 tar 파일을 이동합니다. 예를 들면 '/home/username/e1000' 또는 '/usr/local/src/e1000'을 사용하십시오.

  2. 이동한 tar 파일의 압축을 풉니다. <x.x.x>는 드라이버 tar 파일의 버전 번호입니다.

    tar zxf e1000-<x.x.x>.tar.gz

  3. 드라이버 src 디렉토리로 이동합니다. <x.x.x>는 드라이버 tar의 버전 번호입니다.

    cd e1000-<x.x.x>/src/

  4. 드라이버 모듈을 컴파일합니다.

    make install

    다음과 같이 이진 파일이 설치됩니다.

    /lib/modules/<kernel_version>/kernel/drivers/net/e1000.o

    위에 나열된 설치 위치는 기본 위치이며 Linux 버전에 따라서는 이 위치가 다를 수도 있습니다. 자세한 내용은 ldistrib.htm을 참조하십시오.

  5. 모듈 설치:

    insmod e1000 <parameter>=<value>

  6. 다음을 입력하여 인터페이스에 IP 주소를 할당합니다. <x>는 인터페이스 번호입니다.

    ifconfig eth<x> <IP_address>

  7. 다음을 입력하여 인터페이스가 작동하는지 확인하십시오. <IP_address>는 인터페이스를 테스트할 때 같은 서브넷에 있는 다른 시스템의 IP 주소입니다.

    ping <IP_address>

기타 운영 체제

다른 드라이버를 설치하려면 어댑터 CD에 있는 사용자 안내서 또는 고객 지원 웹사이트( http://support.intel.com)를 참조하십시오.
