Microsoft* Windows NT* 4.0 Push Install Instructions

The following document provides instructions for a basic unattended installation of Microsoft* Windows NT*  that includes the installation of drivers for the Intel® PRO/100 and Intel® PRO/1000 adapters. A "Push", or unattended installation of Windows NT 4.0 provides a means for network administrators to easily install Windows NT 4.0 on systems that have similar equipment.

For more information and instructions, see the pushnt4.txt file in the \APPS\SETUP\PUSH\WINNT directory on the Product CD or in the download directory.
