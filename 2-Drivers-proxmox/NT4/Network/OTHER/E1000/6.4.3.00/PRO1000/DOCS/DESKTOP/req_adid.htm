†LED = The number of LEDs visible on the adapter. If a number is not indicated for the product, it is a LAN On Motherboard device (LOM). In other words, the device resides on the motherboard of the system. Since the vendor configures the LEDs, the number of LEDs visible varies from one vendor to another.
