Install the Driver

Before installing the driver, insert the adapter in the computer and plug in the cable. When Microsoft Windows XP discovers the new adapter, it attempts to find an acceptable Windows driver already installed with the operating system.  One of the following happens at this point:

Windows XP finds a Windows driver and installs this driver without any user intervention. However, this Windows driver may not be most current one. This scenario is called Installation Scenario A. If you do not see the Found New Hardware Wizard window, follow the instructions below for installation scenario A.

Windows XP does not find a Windows driver and displays the Found New Hardware Wizard window. This scenario is called Installation Scenario B. If you see the Found New Hardware Wizard window, follow the instructions below for installation scenario B.

Installation Scenario A

Windows XP finds an acceptable Windows driver that provides basic functionality. The driver installation occurs without prompting you for any information or action. 

At this point, it is important to update the driver. Updating the driver ensures that you have access to all the adapter features.

Installation Scenario B

Windows XP cannot find an acceptable Windows driver. The Found New Hardware Wizard window displays.

  1. If you are installing drivers from the Product CD, insert the CD. If you do not have the CD, download drivers from the support website and transfer them to the system.

  2. In the Found New Hardware Wizard screen, click Next.

  3. The necessary files are copied to your computer. The wizard displays a Completed message. 

  4. Click Finish.

  5. If you want to use the Intel® PROSet software to configure advanced settings, set up teams and VLANs, and use other features, start the Autorun located on the CD. Click Install Software on the Autorun screen.

If Windows does not detect the adapter, follow the instructions below.

  1. Download the drivers and software from the support website. Transfer these downloaded files to the system.

  2. In the Found New Hardware Wizard screen, choose the option to search in a specific location. Click Next.

  3. Enter the path to the location of the downloaded files. Click Next. The necessary files are copied to your computer. The wizard displays a Completed message.

  4. Click Finish.

Installing Additional Adapters

To install additional adapters or more than one adapter, follow the instructions below.

  1. Insert the Product CD. The Autorun program starts. 

    If you downloaded drivers from the support website, browse to the directory containing the downloaded files and run autorun.exe.

  2. Click Install Base Driver. A message telling you that a reboot is required on Windows 98 and Windows NT 4.0 systems.  

  3. Click OK to install all base drivers.

Advanced Settings for an Adapter

To view or change advanced settings for your driver/adapter, use Intel PROSet. Advanced settings vary by adapter, and are described in the Intel PROSet Help. 

CAUTION: Only expert users or network administrators should change advanced settings. You may severely affect your adapter's operation by changing a parameter to a value that is not compatible with your network.

Update the Driver

You can update drivers using either the Product CD or the Update Device Driver Wizard.

  • If you use the Product CD, installation is a one-step process. You can install drivers and Intel PROSet at the same time.

  • If you use the Hardware Update Wizard to install drivers, you must first install the drivers and then install other software components such as Intel PROSet.

Updating with the Product CD

First, install the network adapter in the computer, connect the cable, plug in the power cord and start the computer. When Windows starts the Found New Hardware Wizard, cancel the Wizard. Follow these instructions:

  1. Insert the Product CD. The autorun program starts. 

    If you downloaded drivers from the support website, browse to the directory containing the downloaded files and run autorun.exe.

  2. Click Install Base Driver. A message telling you that a reboot is required on Windows 98 and Windows NT 4.0 systems. 

  3. Click OK to install all base drivers.

Updating with the Hardware Update Wizard 

  1. Right-click My Computer and select Manage. The Microsoft Management Console (MMC) window displays.

  2. In the left pane, select Device Manager. A list of devices appears in the right pane. 

  3. In the right pane, open the Network adapters category. A list of network adapters appears.

  4. Locate the Intel adapter. Right-click it and select Update Driver from the drop-down menu. The Hardware Update Wizard starts.

  5. Select Install the Software Automatically (Recommended). Click Next.

  6. Click Finish.

  7. If you want to use the Intel PROSet software to configure advanced settings, set up teams and VLANs, and use other features, start the autorun located on the CD. Click Install Software on the Autorun screen.

If you are installing a driver in a computer with existing Intel adapters, be sure to update all the adapters and ports with the same new software. This procedure ensures that all adapters function correctly.

Removing the Adapter

If you perform these instructions out of order, it is likely that you will encounter a phantom adapter. The driver and registry association exist, but there is no physical adapter.

Remove the Adapter from a Team

  1. Open Intel PROSet.

  2. If the adapter is part of a team, right-click the adapter name and select Remove from Team. 

  3. If the adapter is part of a VLAN, right-click the VLAN listing and select Remove VLAN.

  4. Right-click the adapter you wish to remove.
    NOTE: If this step is not performed, upon restart this will result in a "phantom adapter" (driver, registry association, but no physical adapter). 

  5. From the drop-down menu, select Remove Adapter.

  6. Click OK to close Intel PROSet. 

Uninstall Intel PROSet

You do not need to uninstall Intel PROSet unless you are permanently removing all Intel adapters. You uninstall Intel PROSet using either Add-Remove Programs or the Product CD.

Uninstalling Intel PROSet with the Product CD

  1. Insert the Product CD. The Autorun program starts. 

  2. Click Install Software. The InstallShield Wizard screen displays.

  3. Click Next. The Program Maintenance dialog displays.

  4. Click Modify. The Custom Setup dialog displays.

  5. Click the icon next to the feature you want to remove. From the pop-up dialog, click Do not install this feature. Click Next.

  6. Click Install.

  7. Click Finish.

The installer removes Intel PROSet from the computer. 

Uninstalling with Add-Remove Programs 

  1. From the Control Panel, double-click Add-Remove Programs. The Add-Remove Programs window displays.

  2. Select Intel(R) PROSet. Click the Remove button. 

  3. When the confirmation message displays, click Yes. 

  4. Allow the uninstall process to complete. 

Uninstall the Driver

You should uninstall the driver if you are permanently removing Intel adapters or if you need to perform a clean installation of new drivers.

  1. While the Add-Remove Programs dialog is open, select Intel PRO Ethernet Adapter and Software. Click Change/Remove. 

  2. When the confirmation message displays, click Yes.

  3. If prompted, restart the computer. 

Tips for Installing Drivers More Easily in the Future

When you install drivers next time, follow the instructions below instead of the instructions for scenario A and B. They make driver installation easy.

NOTE: DO NOT install the adapter hardware yet. Wait until instruction 4 before installing the hardware.

  1. DO NOT install the adapter hardware.

  2. Insert the Product CD. The Autorun program on the CD starts. 

    If you downloaded drivers from the support website, browse to the directory containing the downloaded files and run autorun.exe.

  3. If you want to install Intel PROSet when you install the driver, click Install Software. Otherwise, click Install Base Driver. A message box appears asking if you want to continue. You can use Intel PROSet software to configure advanced settings, set up teams and VLANs, and use other features.

  4. Click OK. The current driver for the adapter is installed on your hard drive.

  5. Shut down the computer and install the adapter hardware.

  6. Power up the computer. The adapter is automatically installed with the correct driver.
