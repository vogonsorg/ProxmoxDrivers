Microsoft* Windows* 2000 Push Install Instructions

The following document provides instructions for a basic unattended installation of Microsoft* Windows* 2000 that includes the installation of drivers for the Intel® PRO/100 and Intel® PRO/1000 adapters. A "Push," or unattended installation of Windows 2000 provides a means for network administrators to easily install the operating system on systems that have similar equipment.

For more information and instructions, see the pushw2k.txt file in the APPS\SETUP\PUSH\WIN2K directory on the Product CD.
