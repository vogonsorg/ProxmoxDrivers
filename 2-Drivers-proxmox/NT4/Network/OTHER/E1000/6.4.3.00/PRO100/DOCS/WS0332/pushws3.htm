Push Install Instructions for Microsoft*Windows Server* 2003

The following document provides instructions for a basic unattended installation of Microsoft* Windows Server* 2003 that includes the installation of drivers for the Intel® PRO/100 and Intel® PRO/1000 adapters. A "Push", or unattended installation of Windows Server 2003  provides a means for network administrators to easily install Windows Server 2003 on systems that have similar equipment.

For more information and instructions, see the pushws3.txt file in the APPS\SETUP\PUSH\WS03XP32 directory on the Product CD.
