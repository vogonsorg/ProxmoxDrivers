Alert Standard Format (ASF) Support

ASF is a standards-based technology that allows the IT administrator to manage network nodes without regard to system state. The Distributed Management Task Force (DMTF) sponsored initiative allows the IT administrator to not only identify platform events but also provides the ability to securely perform corrective actions that address these alerts. Further, ASF allows the IT administrator to manage client systems in both the OS present and OS absent state.

If you are not a network administrator and would like more information, contact your network administrator. If you are a network administrator and want more information, contact the vendor or consult the documentation shipped by the vendor.
