Intel Network Drivers for Windows* 98 (Second Edition) and Windows Millennium (Me)

  • Windows 98 Second Edition (SE)

  • Windows Millennium (Me)

Windows 98 Second Edition (SE)

Before installing the drivers, make sure you have inserted the adapter in the computer and plugged in the network cable. Intel® PRO/100 and Intel® PRO/1000 Server adapters are not supported on Windows 98.

When Windows 98 discovers the new adapter, the Found New Hardware Wizard automatically starts.

  1. If you are installing drivers from the Product CD, insert the CD in the CD-ROM drive. If the Autorun program on the CD displays, ignore it.

  2. In the Add New Hardware Wizard window, click Next.

  3. Choose the Search option and click Next.

  4. Click the Specify Location checkbox and browse to the PRO100\WIN_98ME directory on the CD or from the download directory. Click OK.

  5. Click Next.

  6. Windows 98 searches for a driver. When the search is complete, a message indicates that a driver was found. Click Update driver.

  7. Click Next.

  8. Click Next again.

  9. The necessary files are copied to your computer. The wizard displays a Completed message. 

  10. Click Finish.

Installing Additional Adapters

When you use the Found New Hardware Wizard to install drivers, Windows 98 installs the driver for the first adapter and then automatically installs additional drivers.

Advanced Settings for Windows

To view or change advanced settings for your driver/adapter, use the Intel® PROSet II software. Advanced settings vary by adapter, and are described in the Intel PROSet II Help. For instructions on installing Intel PROSet II, click here.

CAUTION: Only expert users or network administrators should change advanced settings. You may severely affect your adapter's operation by changing a parameter to a value that is not compatible with your network.

Updating the Driver in Windows 98

You can update drivers using either the Product CD or the Update Device Wizard.

Updating with the Product CD

You can update drivers using the Autorun program on the Product CD. 

  1. Insert the Product CD. The Autorun program starts. 

    If you downloaded drivers from the support website, browse to the directory containing the downloaded files and run autorun.exe.

  2. Insert the Product CD. The autorun program starts. 

    If you downloaded drivers from the support website, browse to the directory containing the downloaded files and run autorun.exe.

  3. Click Install Base Driver. A message telling you that a reboot is required on Windows 98 and Windows NT 4.0 systems. 

  4. Click OK to install all base drivers.

  5. If you want to install Intel PROSet II, click Install Software in the autorun screen. You can use Intel PROSet II software to configure advanced settings, set up teams and VLANs, and use other features.

  6. Follow the instructions on the screen.

  7. Restart the computer when prompted.

Updating with the Update Device Driver Wizard

  1. From the Control Panel, double-click the System icon and click the Device Manager tab.

  2. Double-click Network Adapters in the list area.

  3. Double-click the adapter.

  4. Click the Driver tab.

  5. Click Update Driver.

  6. Click Next.

  7. Click the circle next to the "Display a list of the drivers in a specific location...". Do not select "Search for a better driver...". Click Next.

  8. Click Have Disk.

  9. If you have the Product CD, insert it when prompted.

  10. Click the Specify Location checkbox and browse to the PRO100\WIN_98ME directory from the CD or from the download directory. Click OK.

  11. Highlight the Intel adapter and click OK.

  12. Click Next.

  13. Click Finish.

  14. Restart the computer.

If you are installing a driver in a computer with existing Intel adapters, be sure to update all the adapters and ports with the same new software.  Follow this procedure for each adapter your are installing.  This procedure ensures that all adapters function correctly.

For troubleshooting information, click Troubleshooting in the Table of Contents.

Manually Adding an Adapter in Windows 98

If the Plug-and-Play software does not recognize the adapter, you can manually add the adapter.

  1. Right-click Network Neighborhood and select Properties.

  2. Click Add.

  3. Select Adapter.

  4. Click Add.

  5. Select Intel as the manufacturer and then select the adapter type.

  6. Click OK.

  7. When prompted, restart the computer.

Installing Intel PROSet II

  1. Insert the Product CD. The Autorun program starts. 

    If you downloaded drivers from the support website, browse to the directory containing the downloaded files and extract them. The autorun automatically starts.

  2. Click Install Software. A message telling you that a reboot is required on Windows 98 and Windows NT 4.0 systems. Click OK.

  3. Reboot the computer.

 You can use Intel PROSet II software to configure advanced settings, set up teams and VLANs, and use other features.

Removing Adapters and Software in Windows 98

This section describes how to:

  • Uninstall Intel PROSet II

  • Uninstall the driver

  • Remove an adapter

NOTE: If your computer is a hot plug-capable system, see your computer documentation for special removal instructions.
CAUTION: Removing an adapter results in a disruption of all network traffic through that adapter.

Uninstall Intel PROSet II

You do not need to uninstall Intel PROSet II unless you are permanently removing all Intel adapters. You can uninstall Intel PROSet II using either Add-Remove Programs or the Product CD.
NOTE:  Before uninstalling Intel PROSet II, you must remove teams and VLANs configured in Intel PROSet II. 

Uninstalling Intel PROSet II with the Product CD

  1. Insert the Product CD. The Autorun program starts. 

  2. Click Install Software. The InstallShield Wizard window displays.

  3. Click Next. The Program Maintenance dialog displays. 

  4. Click Modify. The Custom Setup dialog displays.

  5. Click the icon next to the feature you want to remove. From the pop-up dialog, click Do not install this feature. Click Next.

  6. Click Install.

  7. Click Finish. Restart the computer when prompted.

The installer removes Intel PROSet II from the computer. 

Uninstalling with Add-Remove Programs 

  1. From the Control Panel, double-click Add-Remove Programs. The Add-Remove Programs window displays.

  2. Select Intel(R) PROSet. Click the Add/Remove button. The Autorun displays.

  3. Click Install Software. The InstallShield Wizard window displays.

  4. Click Next. 

  5. Click Modify. The Custom Setup dialog displays.

  6. Click the icon next to the feature you want to remove. From the pop-up dialog, click Do not install this feature. Click Next.

  7. Click Install.

  8. Click Finish. If prompted, restart the computer.

The Intel PROSet entry is removed from the Add/Remove Programs window.

Uninstall the Driver

You should uninstall the driver if you are permanently removing Intel adapters, or if you need to perform a clean installation of new drivers.

Before you uninstall the driver:

  • Make sure the adapter is not a member of a team. If the adapter is a member of a team, remove the adapter from the team in Intel PROSet II. 
  • Uninstall Intel PROSet II.

If you perform these instructions out of order, it is likely that you will encounter a phantom adapter. The driver and registry association exist, but there is no physical adapter.

To uninstall the driver:

  1. From the Control Panel, double-click Add/Remove Programs. 

  2. Select Intel PRO Ethernet Adapter and Software. 

  3. Click Add/Remove. 

  4. When the confirmation dialog displays, click OK. 

  5. Click Yes to restart the computer immediately or click No to defer computer restart.

Remove Existing Adapters in Windows 98

When you remove an adapter using the instructions below, you also remove the driver.

  1. Double-click My Computer.

  2. Double-click Control Panel.

  3. Double-click System.

  4. Click the Device Manager tab.

  5. Double-click Network Adapters.

  6. Select the adapter driver listed below the Network Adapters group and click Remove.

  7. Click OK.

Windows Millennium (Me)

Before installing the drivers, make sure you have inserted the adapter in the computer and plugged in the network cable. Intel® PRO/100 and Intel® PRO/1000 Server adapters are not supported on Windows Me.

You can install drivers for Intel adapters using the Found New Hardware Wizard.

  1. If you are installing drivers from the Product CD, insert the CD in the CD-ROM drive. If the Autorun program on the CD displays, ignore it.

  2. The Add New Hardware window displays.

    If you have an Product CD, click Next.

    If you downloaded drivers from the support website, click the Specify Location checkbox instead, browse to the directory containing the drivers, and click Next.

  3. The necessary files are copied to your computer. The wizard displays a Completed message

  4. Click Finish. Restart your computer when prompted.

  5. Install Intel PROSet II to configure and test the adapter.

Installation Notes for Windows Me

If you are installing Windows Me or if you are upgrading the adapter drivers in a multi-adapter Windows Me system, read the following.

  • When you are installing the Windows Me operating system (not the drivers), Windows Me may attempt to install its own older drivers on your new adapter and Windows Me reports that it was successful. However upon reboot, you will find the adapter does not work. In the Device Manager you will see that the adapter appears under Other Devices instead of under Network Devices. To make the adapter work, you must update the adapter driver with those included on the Product CD or on the support website. The instructions for updating the driver are in the next section.

  • If you have multiple Intel adapters in your Windows Me system, you must be sure to update all of the Intel adapters to the software included with your new card. Failing to update any other Intel adapter in the system causes the new adapter to not function. It appears as though the new adapter did not receive the proper update when you loaded the new driver, but this is NOT the case.

    Windows Me cannot find the new driver while an older driver still exists on another adapter, so all other Intel adapters must be updated to the new driver. Do not attempt to use any drivers already in any system directory.

    If you downloaded drivers from the support website, browse to the directory containing the downloaded files and extract the files. The autorun automatically starts.

  2. Click Install Base Driver.

  3. When a confirmation message displays, click OK.

  4. Restart the computer when prompted.

Updating with the Update Device Driver Wizard

  1. From the Control Panel, double-click System.

  2. Select the Device Manager tab.

  3. Double-click Network adapter to open it and then select the adapter.

  4. Click Properties.

  5. Select the Driver tab and click Update Driver.

  6. Select Automatic Search for a better driver...

  7. If you have the Product CD, insert it when prompted.

  8. In the Wizard window, type the drive letter for the Product CD or specify the download directory. Click Next.

  9. Click Finish.

  10. When prompted, restart the computer.
