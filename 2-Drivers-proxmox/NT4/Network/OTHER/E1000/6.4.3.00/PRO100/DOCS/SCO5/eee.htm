Intel PRO/100 Network Driver for SCO OpenServer*

The Intel PRO/100 adapter driver for SCO OpenServer* is packaged in the eeE.vol media image volume file. To install the eeE.vol package and setup the driver for a network adapter, use the following steps:

  1. Copy the eeE.vol file to any directory, such as /tmp. Rename the file as VOL.000.000, and make the file read-only:

    cp eeE.vol /tmp/VOL.000.000
    chmod 444 /tmp/VOL.000.000

  2. If there is an older version of the driver on the system, you must first remove it. To do this, run netconfig. Remove all instances of the "Intel ..." adapters. Exit netconfig without opting to relink the kernel. Run custom. Remove the older driver for the Intel adapter.

  3. Install the new driver using the custom utility. To open the utility, type custom. When asked for the installation media, choose media images, and type the directory path to the VOL.000.000 file. (In step 1, if you copied it to /tmp, type /tmp). After the installation of the driver is complete, exit custom.

  4. Run netconfig and add the adapters. For each adapter that is present in the system, enter the appropriate TCP/IP parameters. By default, the driver automatically detects the line speed and duplex mode. If you want to force any of these settings, choose Advanced Options and set the speed and duplex modes. Exit netconfig and choose the option to relink the kernel.

  5. Reboot the system.

Other configurable options are available in the Space.c file. Typically, these options do not need to be modified. If any of these variables are changed, the kernel has to be re-linked, and the system must be rebooted for the new settings to take effect.
