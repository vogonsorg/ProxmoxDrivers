Intel® PRO -verkkosovittimen asentaminen

Muutos 04/2004

Intel PCI-, PCI-X-, tai PCI-Express* -sovittimen asentaminen

  1. Sammuta tietokone ja irrota virtajohto.

  2. Poista tietokoneen kansi ja sovittimen paikan suojus siitä korttipaikasta, joka vastaa sovitintasi.

  3. PCI Express -sovittimet: aseta sovittimen reunaliitin PCI Express -paikkaan ja kiinnitä pidike runkoon.
    PCI- ja PCI-X-sovittimet: aseta sovittimen reunaliitin tukevasti PCI-paikkaan, niin että se on täysin sisällä. Kiinnitä pidike runkoon.

  4. Aseta tietokoneen kansi paikalleen. Kytke virtajohto.

Verkkokaapelin kytkeminen

Intel® PRO/100 -sovittimet ja PRO/1000 -kuparisovittimet: Kytke verkkokaapeli tai kaapelit sovittimeen käyttäen 10 Mbps -verkkoihin tarkoitettua Cat 3-, Cat 4- tai Cat 5 -yhteensopivaa TPE (Twisted Pair Ethernet) -verkkokaapelia. Käytä 100 Mbps- tai 1000 Mbps -verkossa ja kaikissa kotiverkoissa Cat 5 (4-parista) -kaapelia.

Intel® PRO/1000 -kuitusovittimet, joissa SC-liitin: Irrota kuituoptisen liitännän suojus ja säilytä se. Liitä kuituoptinen kaksisuuntainen 1000Base-SX SC -liitin sovitinkiinnikkeen TX/RX-liitäntöihin.

Intel® PRO/1000 -kuitusovittimet, joissa LC-liitin: Irrota kuituoptisen liitännän suojus ja säilytä se. Liitä LC-liitin sovittimeen käyttämällä oikeaa kaapelointia sovittimen tyypin (1000BASE-SX tai 1000BASE-LX) mukaan. Muiden liitintyyppien (kuten SC-liittimen) muuntosovittimia voidaan käyttää silloin, kun kaapelointi vastaa sovittimen optisia määrityksiä, pituusrajoitukset mukaan lukien.

 Käynnistä tietokone ja noudata alla olevia ohjaimen asennusohjeita.

Asenna ohjaimet ja laajennukset

Huomautus: Intel PRO/1000 Server -sovittimet eivät toimi Windows* 95/98/Me -tietokoneissa.

Windows-käyttöjärjestelmät (ei Windows NT* 4.0)

Huomaa, että sinulla on oltava järjestelmänvalvojan oikeudet, jotta voit asentaa ohjaimet.

  1. Sovitin on tunnistettu ja Windows asentaa joko Windowsissa olevan ohjaimen tai käynnistää ohjatun Etsi uusi laite-toiminnon.

  2. Jos ohjattu toiminto käynnistyy, asenna ohjain sen kehotteiden mukaan tämän paketin CD-levystä.

  3. Jos Windows asentaa Windowsissa olevan ohjaimen (ohjattua toimintoa ei tule näkyviin), päivitä ohjain asettamalla Intel CD -levy CD-asemaan ja napsauttamalla Asenna ohjaimet -painiketta, kun Intel PRO -verkkoyhteydet -suoritusohjelman ikkuna tulee näyttöön.

  4. Kun Asennusvaihtoehdot -valintaikkuna tulee näyttöön, valitse Edistyneet verkkopalvelut, jos haluat määrittää ryhmiä tai VLAN-verkkoja.

Windows NT* 4.0

Huomautus: Intel PROSet -apuohjelma asennetaan automaattisesti sovittimen ohjaimen asentamisen yhteydessä.

  1. Käynnistä Windows NT* 4.0, kaksoisnapsauta Ohjauspaneelin Verkko-kuvaketta, valitse Sovittimet-välilehti ja valitse sitten Lisää.

  2. Kun näyttöön tulee sovittimien luettelo, älä valitse sovitinta luettelosta, vaan aseta sovittimen mukana toimitettu CD-levy tietokoneeseen tai osoita muuta tietovälinettä ja valitse Levy-painike.

  3. Jos näyttöön tulee CD-levyn suoritusohjelman ikkuna, sulje se ja jatka.

  4. Määritä polku, valitse OK ja jatka asennusta. Kun toimenpide on valmis, uusi Intel-sovitin näkyy Verkkosovittimet-luettelossa.

  5. Lopeta asennus valitsemalla Sulje.

  6. Käynnistä tietokone uudelleen.

Linux

Voit muodostaa tämän ohjaimen binaarisen RPM*-tiedoston suorittamalla komennon 'rpmbuild -tb <tiedostonimi.tar.gz>'. Korvaa <tiedostonimi.tar.gz> ohjaimen nimellä.

  HUOMAUTUKSIA: Jotta koontikäännös toimisi oikein, parhaillaan toiminnassa olevan ytimen ON vastattava asennettujen ydinlähteiden versiota ja kokoonpanoa. Jos olet juuri kääntänyt ytimen uudelleen, käynnistä järjestelmä nyt uudelleen.

RPM-toiminnot on testattu vain Red Hat -jakeluissa.

Käytä Intel PRO/100 -sovittimien kohdalla ohjaimen nimeä e100.

  1. Siirrä perusohjaimen tar-tiedosto valitsemaasi hakemistoon. Käytä esimerkiksi hakemistoa '/home/username/e1000' or '/usr/local/src/e1000'.

  2. Pura pakattu tiedosto seuraavalla komennolla, jossa <x.x.x> on ohjaimen tar-tiedoston version numero:

    tar zxf e1000-<x.x.x>.tar.gz

  3. Siirry ohjaimen src-hakemistoon seuraavalla komennolla, jossa <x.x.x> on ohjaimen tar-tiedoston version numero:

    cd e1000-<x.x.x>/src/

  4. Käännä ohjainmoduuli seuraavalla komennolla:

    make install

    Binaaritiedosto asennetaan seuraavalla nimellä seuraavaan hakemistoon:

    /lib/modules/<kernel_version>/kernel/drivers/net/e1000.o

    Edellä annetut asennuskohteet ovat oletuskohteita. Ne eivät ehkä vastaa joidenkin Linux-jakeluiden asennuskohteita. Saat lisätietoja siirtymällä tiedostoon ldistrib.htm.

  5. Asenna moduuli seuraavalla komennolla:

    insmod e1000 <parametri>=<arvo>

  6. Määritä liittymän IP-osoite antamalla seuraava komento, jossa <x> on liittymän numero:

    ifconfig eth<x> <IP_osoite>

  7. Varmista, että liittymä toimii. Voit tehdä sen antamalla seuraavan komennon, jossa <IP_osoite> on toisen koneen IP-osoite samassa aliverkossa, jossa liittymää testataan:

    ping <IP_osoite>
