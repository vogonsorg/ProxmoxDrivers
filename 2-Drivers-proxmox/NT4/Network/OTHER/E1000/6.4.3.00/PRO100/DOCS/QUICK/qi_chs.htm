英特尔(R)网络适配器安装

修订版本 04/2004

安装英特尔 PCI、PCI-X、或 PCI-Express* 适配器

  1. 关机并拔出电源线。

  2. 打开计算机机壳以及与您的适配器相配的适配器插槽盖。

  3. 对 PCI Express 适配器：将适配器边缘连接器插入 PCI Express 插槽，将支架固定于机箱。
    对 PCI 和 PCI-X 适配器：将适配器边缘连接器牢固插入 PCI 插槽，使之完全就位。将支架固定于机箱。

  4. 装回机盖。插入电源线。

连接网络电缆

英特尔(R) PRO/100 和 PRO/1000 铜质适配器： 将网络电缆或电缆连接至适配器；对 10 Mbp 网络，使用 3 类、4 类或 5 类双绞线以太网（TPE）网络电缆。对 100 或 1000 Mbp 网络和任何民宅网络，使用 5 类（4 对）电缆。

带 SC 连接器的英特尔(R) PRO/1000 光纤适配器：卸下并保存光纤连接器盖。将 1000Base-SX 双工 SC 型光纤连接器插入适配器支架上的 TX/RX 端口。

带 LC 连接器的英特尔(R) PRO/1000 光纤适配器：卸下并保存光纤连接器盖。将 LC 连接器插入适配器，其布线应与适配器类型相配（1000BASE-SX 或 1000BASE-LX）。如果布线与适配器的光学规格相配，包括长度限制，则可使用其他类型连接器（如　SC）的转换电缆。

安装驱动程序和 扩展

注意：英特尔 PRO/1000 服务器适配器不能在 Windows* 95、Windows 98* 和 Windows Me* 计算机上操作。

Windows 操作系统（Windows NT* 4.0 除外）

注意：必须对操作系统有管理权限方能安装驱动程序。

  1. 检测到适配器时，Windows 或者安装一个 Windows 的驻留驱动程序或者启动发现新硬件向导。

  2. 如果启功了向导，遵照提示安装此程序包中 CD 上的驱动程序。

  3. 如果安装 Windows 的驻留驱动程序（不出现向导），则应更新此驱动程序。方法是：在英特尔 PRO 网络连接 autorun 窗口出现时，插入英特尔 CD，并单击安装驱动程序按纽。

  4. 如果需要设置组或 VLAN， 则在安装选项对话框出现时，选择高级网络服务。

Windows NT 4.0

注意：英特尔 PROSet 工具软件在适配器驱动程序安装过程中自动安装。

  1. 启动 Windows NT 4.0，双击“控制面板”上的“网络”图标，单击“适配器”选项卡，再单击“添加”。

  2. 见到适配器列表时，不要从列表上选择适配器，而是插入随适配器提供的光盘，或指向其他媒体，再单击“从磁盘安装”。

  3. 如果光盘“自动运行”屏幕出现，将它关闭，再继续。

  4. 指定路径, 单击“确定”，继续安装。完成时，新的英特尔适配器将出现在“网络适配器”列表中。

  5. 单击“关闭”完成安装。

  6. 重新启动计算机。

Linux*

要编译该驱动程序的二进制 RPM* 包，运行‘rpmbuild -tb <文件名.tar.gz>’。用该驱动程序的实际文件名代替 <文件名.tar.gz>。

  注意:要使编译版本正确运行，当前运行的内核的版本和配置必须与安装的内核源相符。如果刚重新编译了内核，现在则应重新引导系统。

RPM 功能仅在 Red Hat 发布上测试过。

对英特尔 PRO／100 适配器，驱动程序名称用 e100。

  1. 将基本驱动程序的 tar 文件移往您选择的目录。例如，使用‘/home/username/e1000’或‘/usr/local/src/e1000’。

  2. 解压缩该档案文件，其中 <x.x.x> 是驱动程序压缩文件的版本号：

    tar zxf e1000-<x.x.x>.tar.gz

  3. 切换到驱动程序 src 目录下，其中 <x.x.x> 是驱动程序压缩文件的版本号：

    cd e1000-<x.x.x>/src/

  4. 编译驱动程序模块：

    make install

    二进制文件将安装为：

    /lib/modules/<内核_版本>/kernel/drivers/net/e1000.o

    以上列出的安装位置为默认位置。它们对某些 Linux 发布来说不一定正确。要获得更多信息，请访问 ldistrib.htm。

  5. 安装模块：

    insmod e1000 <参数>=<值>

  6. 用下列命令指派 IP 地址至接口卡，其中 <x> 是接口卡号：

    ifconfig eth<x> <IP 地址>

  7. 验证接口卡正常工作。输入以下命令。其中 <IP 地址>是与被测接口卡位于同一子网的另一台计算机的 IP 地址：

    ping <IP 地址>
