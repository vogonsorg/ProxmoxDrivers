Install the Drivers

Before you begin

To successfully install drivers or software, you must have administrative privileges on the computer completing installation.

To install drivers

You can obtain drivers in one of the following ways:

  • Install them directly from the Product CD.

  • Create a floppy disk and install the drivers from the floppy disk.

  • Download the drivers from the support website and install them from the download directory.

For directions on how to install drivers for a specific operating system, select one in the table of contents.

NOTE: If you are installing a driver in a computer with existing Intel adapters, be sure to update all the adapters and ports with the same driver and Intel® PROSet software. This ensures that all adapters will function correctly.
