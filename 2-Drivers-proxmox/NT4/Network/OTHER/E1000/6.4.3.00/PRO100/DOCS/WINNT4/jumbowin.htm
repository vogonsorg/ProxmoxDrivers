Setting Up Jumbo Frames in Windows

NOTE: Jumbo frames are not supported in multi-vendor team configurations.

  1. Start Intel® PROSet and click the Advanced tab.

  2. Press <F1> and follow the instructions to enable Jumbo Frames.

  3. Set your desired packet size (based on network capability).

  4. Click OK or Apply to save your changes. 

Common frame sizes are displayed, but the entire range (1,500-16,000 bytes) is available. All equipment on the network must also support the larger frame size, and the switch should be set 8 bytes larger than what is specified in Intel PROSet.
