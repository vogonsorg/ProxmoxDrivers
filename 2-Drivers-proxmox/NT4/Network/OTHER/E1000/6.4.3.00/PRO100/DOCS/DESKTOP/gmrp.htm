GMRP Support

GMRP (GARP Multicast Registration Protocol) allows computers to subscribe to a GARP ( Generic Attribute Registration Protocol) multicast group. GARP enables switches to send multicast frames only to members of a multicast group, instead of flooding the entire network with multicasts, helping improve the performance of the network. GMRP and GARP are industry-standard protocols, defined in the IEEE 802.1d and 802.1p specifications.

If the network uses GARP multicasting, enable GMRP to allow this computer to subscribe to the GARP multicast group.

For more information, select Intel (R) PROSet in the Table of Contents (left pane) of this window.

NOTE: You can enable GMRP support on Teams and individual adapters. Single adapters must use a VLAN to enable GMRP.

Requirement

For GMRP to be supported across the network, the network must be using GARP.
