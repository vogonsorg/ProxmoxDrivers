PCI Hot Plug Support for Microsoft* Windows* 2000, XP, and Windows Server 2003 Server

Intel® PRO/100 and PRO/1000 network adapters are enabled for use in selected servers equipped with PCI Hot Plug support and running Microsoft* Windows 2000, Windows XP, and Windows Server 2003.

PCI Hot Plug only works when you hot plug an identical Intel network adapter.

NOTE: If you have a teamed adapter in a PCI Hot Plug slot and must replace it, do not insert a replacement adapter into the same network unless the server has been rebooted. The two adapters may have the same virtual MAC address and create a conflict.

For more information on setting up and using PCI Hot Plug support in your server, see your hardware and/or Hot Plug support documentation for details.
