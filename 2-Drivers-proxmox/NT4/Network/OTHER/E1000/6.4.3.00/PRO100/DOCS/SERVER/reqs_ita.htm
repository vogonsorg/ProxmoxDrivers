System Requirements

Before installing the network adapter, check your system for the following minimum configuration requirements for adapters in Intel® Itanium™-based architectures.

Supported Operating Systems

The Intel drivers for Itanium-based systems operate with the following operating systems:

  • Microsoft Windows* XP 64-Bit Edition

  • Linux 2.2.x or 2.4.x
