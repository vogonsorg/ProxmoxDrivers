Setting Up ANS Teaming in Microsoft* Windows*

In Windows, use Intel® PROSet to set up ANS teams. For instructions, refer to the help included with Intel PROSet. For more information, select Intel (R) PROSet in the Table of Contents (left pane) of this window.

NOTE: ANS Teaming is compatible with Microsoft Network Load Balancing (NLB/WLBS) running on Microsoft* Windows 2000 and Windows Server 2003 when in multicast mode only. To use Microsoft Network Load Balancing  in an Adaptive Load Balancing (ALB) team mode, Receive load balance must be disabled.
NOTE: Intel® Itanium®-based systems support Windows Server 2003 for 64-bit systems only.
