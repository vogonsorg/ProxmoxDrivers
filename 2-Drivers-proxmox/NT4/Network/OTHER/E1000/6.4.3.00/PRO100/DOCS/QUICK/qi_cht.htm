Intel(R) PRO 網路介面卡安裝

修訂日期 04/2004

安裝 Intel PCI、PCI-X,、或 PCI-Express* 介面卡

  1. 關掉電腦並拔下電源線。

  2. 移除電腦蓋，並從符合介面卡的插槽移除介面卡插槽。

  3. 若是 PCI Express 介面卡： 將介面卡邊緣接頭插入 PCI Express 插槽，並將支架固定到底座。
    若是 PCI 和 PCI-X 介面卡： 穩固地將介面卡邊緣接頭放入 PCI 插槽，直到其完全插入為止。 將支架固定到底做。

  4. 將電腦外殼裝回原位。 插入電源線。

接上網路纜線

Intel(R) PRO/100 和 PRO/1000 銅式介面卡：在 10 Mbp 的網路使用類別 3、4 或 5 "雙絞線乙太網" (TPE) 網路纜線將網路懶現貨纜線連接到介面卡。 在 100 或 1000 Mbps 網路和任何常駐型網路使用 “類別 5”纜線 (4 對)。

含 SC 接頭的 Intel(R) PRO/1000 纖維介面卡：移除並存放光纖接頭護蓋。 將以 1000 為基礎的 SX 雙用類型 SC 光纖接頭插入配接卡框架上的 TX/RX 連接埠內。

含 LC 接頭的 Intel(R) PRO/1000 纖維介面卡：移除並存放光纖接頭護蓋。 將適用於您的配接卡類型的正確纜線的 LC 接頭插入配接卡（1000BASE-SX 或 1000BASE-LX)。 如果纜線的規格與配接卡相符 (包括長度規格)，則可考慮將纜線換成其它接頭類型 (例如 SC)。

啟動電腦並遵照以下的驅動程式安裝說明執行。

安裝驅動程式和延伸功能

注意： "Intel PRO/1000 伺服器介面卡" 無法在 Windows* 95、Windows 98* 和 Windows Me* 電腦中作業。

Windows 作業系統 (Windows NT* 4.0 除外)

請注意，您一定要有管理權利才能安裝驅動程式。

  1. 偵測到介面卡後，Windows 會安裝一個 Windows 常駐驅動程式或啟動"尋找新增硬體" 精靈。

  2. 精靈啟動時，請遵照提示從封裝中的光碟片安裝驅動程式。

  3. 如果安裝了 Windows 常駐驅動程式的話 (精靈沒有出現)，請插入 Intel CD 光碟片，在Intel PRO Network Connections自動執行視窗出現時，按一下「安裝驅動程式」來更新驅動程式。

  4. 當「安裝選項」對話方塊出現時，如果您要設定群組或 VLAN 的話，請選擇「進階網路服務」。

Windows NT 4.0

注意： 注意：Intel PROSet 公用程式軟體會在介面卡驅動程式安裝期間自動被安裝。

  1. 啟動 Windows NT 4.0，連按兩下「控制台」上的「網路」圖示，然後按一下「介面卡」標籤，再按一下「新增」。

  2. 當您看到介面卡的清單時，請不要從清單選擇介面卡，請將隨介面卡提供的光碟片插入或指到其它媒體，，然後按一下「從磁片安裝」。

  3. 如果 CD Autorun（光碟片自動執行）螢幕出現的話，請將其關閉，然後再繼續進行。

  4. 請指定路徑，然後按一下「確定」，之後再繼續進行安裝。 安裝完成後，新的 Intel 介面卡就會顯示在「網路介面卡」清單上。

  5. 按一下 Close（關閉）以完成作業。

  6. 重新啟動您的電腦。

Linux*

要建立這個驅動程式的二進位 RPM* 套裝軟體，請執行 'rpmbuild -tb <filename.tar.gz>'。 用特定的驅動程式檔案名稱取代 <filename.tar.gz>。

  注意： 若要使這個建置版本能夠適當作業，目前執行中的核心 "一定" 要與安裝的核心來源的版本和組態相符。 如果您剛重新編譯核心，請馬上重新啟動系統。

RPM 的功能性僅在 Red Hat 傳遞版本中經過測試。

若是 Intel PRO/100 介面卡，請使用 e100 為驅動程式名稱。

  1. 將基礎驅動程式 tar 檔案移到您選擇的目錄。 例如，使用 '/home/username/e1000' 或 '/usr/local/src/e1000'。

  2. 解除 tar/解壓縮該保存檔，此處的 <x.x.x> 是驅動程式 tar 檔案的版次號碼：

    tar zxf e1000-<x.x.x>.tar.gz

  3. 變更驅動程式 src 目錄，此處的 <x.x.x> 是驅動程式 tar 的版次號碼：

    cd e1000-<x.x.x>/src/

  4. 編譯驅動程式模組：

    進行安裝

    二進位安裝如下：

    /lib/modules/<kernel_version>/kernel/drivers/net/e1000.o

    上列的安裝位置是預設位置。 對某些特定的 Linux 傳遞版本來說，這些位置可能不正確。 若須更多資訊，請蒞臨 ldistrib.htm。

  5. 安裝模組：

    insmod e1000 <parameter>=<value>

  6. 輸入以下的指令來指派 IP 位址給介面，此處的 <x> 是介面編號：

    ifconfig eth<x> <IP_address>

  7. 確認介面可以作業。 輸入以下的指令，此處的 <IP_address> 是相同子遮罩上另外一部測試介面之電腦的 IP 位址：

    ping <IP_address>

其它作業系統

要安裝其它驅動程式，請在介面卡 CD 光碟片上或參閱《使用者指南》或是，蒞臨 Intel 的客戶支援網站： http://support.intel.com。
