Intel(R) PRO Network Adapter Driver INF Notes
=============================================


July 1, 2004


This release includes separate INF files for adapters that meet current
Microsoft WHQL-certification requirements, and obsolete 10/100 adapters 
that cannot meet current certification requirements.


- For current adapters, use:

    E100b325.inf.

- For obsolete adapters that do not meet certification requirements, use:

    E100ant5.inf - for Windows* 2000
    E100a325.inf - for Windows XP and Windows Server 2003 (32-bit).

For more information, visit the following web URL:

    http://www.intel.com/support/network/sb/cs-010579.htm
