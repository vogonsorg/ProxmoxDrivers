Manual Installation ->

The files "storport.sys" and "ntoskrn8.sys" must be copied FROM the folder ".\1" INTO
the folder "C:\WINDOWS\system32\drivers". And after that the "Kernel-Mode Driver Framework 1.11" in ".\2" needs to
be installed. And after that the Microsoft USB 3.0 xHCI ".\3" driver can be installed.
